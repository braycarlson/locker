# locker
A peripheral locker to lock/unlock the mouse and keyboard on Windows.

# Keyboard Shortcut
- **Lock:** <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>L</kbd>
- **Unlock:** <kbd>U</kbd> + <kbd>N</kbd> + <kbd>L</kbd> + <kbd>O</kbd> + <kbd>C</kbd> + <kbd>K</kbd>
